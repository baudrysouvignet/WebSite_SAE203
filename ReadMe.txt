Pour vous connecter, vous devez modifier le code dans la section de connexion avec les paramètres de la base de données.

Le nom dépend du nom avec lequel vous avez importé la base de données. Le mot de passe doit être laissé vide si vous êtes sur un Mac et doit être "root" si vous êtes sur Windows.

En ce qui concerne JavaScript, la première interaction se trouve sur la page index.php et correspond au fichier carrousel.js. La deuxième interaction se trouve sur la page account.php une fois connecté et correspond au fichier filtreliste.js. Les informations de connexion sont les suivantes : nom => "test", prénom => "test", mdp => "superest123".

En ce qui concerne le CSS, une partie du code pour faciliter la notation et la compréhension des principales intégrations se trouve dans le fichier cssavance.css, mais il y a également de nombreuses autres interactions dans les autres fichiers CSS.
